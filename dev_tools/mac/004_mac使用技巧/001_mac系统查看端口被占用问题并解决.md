# Mac 系统查看端口被占用的问题

Mac系统通过lsof 命令查看端口被占用情况，相关命令如下
```shell
	$> lsof -i:端口号
```

执行该命令之后，可以看到指定端口被那个应用占用，并且会显示相关的进程号。
之后就可以通过kill 命令指定进程号来杀死指定的进程。命令如下

```shell
	$> kill 进程号
```
